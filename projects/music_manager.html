Music Manager

  • GitHub repository
  • Language(s): C++
  • Libraries: Qt, TagLib
  • Database: SQLite

Features

  • Supports MP3, AAC, WAV, AIFF, FLAC, and ALAC files
  • Add & delete music files
  • Edit tracks & metadata
  • Displays all tracks, albums, & artists
  • SQLite database
  • Qt GUI
  • Developed a full-stack C++ application for managing digital audio files with a Qt-based GUI for intuitive browsing and editing
  • Integrated a SQLite database for persistent storage, enabling more comprehensive data handling beyond standard file metadata
  • Employed TagLib for cross-format audio metadata extraction and editing
